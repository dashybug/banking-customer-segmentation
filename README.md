# banking-customer-segmentation
A data analytics project utilizing K-Means clustering to develop distinct customer profiles within a banking dataset. This analysis identifies behavioral segments by processing transaction patterns and feedback metrics, providing actionable insights into customer engagement and value.

Data-Driven Profiling: Employed K-Means clustering to transform raw customer data into actionable behavioral segments.

Statistical Validation: Used the Elbow Method to statistically determine the appropriate number of clusters for the dataset.

Segment Visualization: Applied PCA (Principal Component Analysis) as a visualization tool to map high-dimensional profile data into a readable 2D format.

Profile Characterization: Defined specific customer personas by analyzing the mean attributes (spend, frequency, satisfaction) of each identified cluster.
